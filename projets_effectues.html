Stage 2e année BTS SIO
J'ai effectué un Stage à Nevers Agglomération pour ma seconde année de BTS SIO
Le but de ce stage était de créer une application web avec le framwork Symfony au format mobile first.
Le besoin de cette application est dû au problème suivant, il est impossible d'utiliser les logiciels OSTickets et GLPI depuis un téléphone portable. J'ai donc créé cette application afin de pouvoir utiliser ces logiceils depuis n'import quel lieu d'intervention.
